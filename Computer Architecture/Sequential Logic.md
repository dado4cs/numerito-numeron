# Sequential Logic
Sequential logic is a type of digital circuit whose output depends not only on the current inputs but also on the history of previous inputs (the *state*).

## Key Characteristics
- It contains **memory** or storage elements.
- It uses a **clock** to synchronize state transitions.
- The **state** is stored in devices like [[D Flip-Flop|flip-flops]] or [[SR Latch|latches]].

## Components
- **Latches:** Level-sensitive storage elements.
- **Flip-flops:** Edge-sensitive storage elements.
- **Registers:** Groups of flip-flops that store multiple bits.
- **Counters:** Circuits that cycle through a sequence of states.
