#architecture #circuit 
## Definition
A digital circuit is an electronic system that processes information using discrete values (0 and 1) known as [[Binary system|bits]]. It is a specific level of [[Abstraction Levels|abstraction]] with higher-level logic.
## Fundamental composition
Digital circuits are constructed using three fundamental building blocks: **AND**, **OR** and **NOT** gates. The logic behind this gates, is based on [[Boolean Algebra]].

### AND Gate
The output is 1 only if all its inputs are 1.
![[and-gate.png]]
## OR Gate
The output is 1 if at least one input is 1.
![[or-gate.png]]
### NOT Gate
Output is the input's complement.
![[not-gate.png]]

## Other gates
Other common two-input logic gates are:
- **XOR (Exclusive OR):** Is true if A or B, but not both are true.
- **NAND:** Performs NOT AND. Its output is true unless the both inputs are true.
- **NOR:** Performs NOT OR. Its output is TRUE if neither A nor B is true.
