## URI
Identificador genérico que se refiere a **cualquier recurso**. Incluye URLs y URNs. Busca identificar un recurso de manera única.

```
URI 
|
|- - URL
|- - URN
```

- URI: identificador general del curso
- URL: URI que indica **dónde está el recurso**
- URN: URI que **solo nombra el recurso**, no dice dónde encontrarlo

## Endpoint
Es una URL específica de una API donde además, se especifica el método:

```
Base URL : https://api.myapp.com
|
|- GET https://api.myapp.com
|- GET https://api.myapp.com/10
|- POST https://api.myapp.com/users
|- DELETE https://api.myapp.com/users 
```
