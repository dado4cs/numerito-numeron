# Recommended C++ Books

## 1. C++ Primer (5th Edition)
- **Authors:** Stanley B. Lippman, Josée Lajoie, Barbara E. Moo.
- **Focus:** The best comprehensive guide for learning modern C++ (C++11/14). It covers basics to advanced template and memory management.

## 2. Effective Modern C++
- **Author:** Scott Meyers.
- **Focus:** Best practices for C++11 and C++14. It covers lambda expressions, rvalue references, move semantics, and more.

## 3. The C++ Programming Language
- **Author:** Bjarne Stroustrup (the creator of C++).
- **Focus:** A definitive guide for experienced programmers, providing insight into the design goals and features of the language.

## 4. Programming: Principles and Practice Using C++
- **Author:** Bjarne Stroustrup.
- **Focus:** An introduction to programming using C++, focusing on software engineering principles and good programming practices.
